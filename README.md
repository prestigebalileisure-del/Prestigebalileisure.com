# Prestige Bali Leisure - Complete CMS System

A comprehensive Content Management System for luxury leisure and tour company in Bali.

## Features

✅ **Admin Dashboard** - Manage all content
✅ **Villa Management** - Create, edit, delete luxury villas
✅ **Tour Management** - Complete tour and experience management
✅ **Multi-step Booking System** - Guest reservations with email notifications
✅ **Mobile App** - iOS & Android support (React Native)
✅ **Strapi CMS Integration** - Headless CMS backend
✅ **Email Notifications** - Booking confirmations and admin alerts
✅ **Responsive Design** - Works on all devices

## Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Strapi Cloud account

### Installation

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run development server
npm run dev
